⚙️ Cara Kerja PDA Penyeimbang Kurung

Pushdown Automaton (PDA) adalah mesin automata yang memiliki memori tambahan berupa stack. Di simulator ini, PDA dirancang untuk mengenali pasangan tanda kurung seimbang:

  • Ketika melihat kurung buka (, kita melakukan Push simbol X ke stack. Ini merepresentasikan kurung buka yang sedang menunggu pasangannya.
  • Ketika melihat kurung tutup ), kita melakukan Pop satu simbol X dari stack. Ini merepresentasikan pasangan kurung buka yang berhasil dipasangkan dan ditutup.
  • Jika melihat kurung tutup ) namun stack kosong (tidak ada X), artinya kurung tutup tidak memiliki pembuka. Simulator langsung menolak (REJECT).
  • Di akhir pita input, jika stack berhasil kembali ke simbol dasarnya yaitu Z0, PDA bertransisi ke state final qf dan menerima input (ACCEPT).

📐 Definisi Formal Mesin (Tuple)

PDA $M = (Q, \Sigma, \Gamma, \delta, q_0, Z_0, F)$ didefinisikan sebagai:

  • $Q = \{q_0, q_f\}$ (Kumpulan state)
  • $\Sigma = \{(, )\}$ (Alfabet input)
  • $\Gamma = \{X, Z_0\}$ (Alfabet stack)
  • $q_0 = q_0$ (State awal)
  • $Z_0 = Z_0$ (Simbol dasar stack)
  • $F = \{q_f\}$ (State penerima/final)

Fungsi Transisi $\delta$:

  • $\delta(q_0, (, Z_0) = (q_0, XZ_0)$
  • $\delta(q_0, (, X) = (q_0, XX)$
  • $\delta(q_0, ), X) = (q_0, \epsilon)$
  • $\delta(q_0, \epsilon, Z_0) = (q_f, Z_0)$
